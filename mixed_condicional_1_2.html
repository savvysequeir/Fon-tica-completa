II. Explicación

Mixed Conditional 1

Uso: una condición del pasado produce un resultado en el presente.

If + sujeto + had + participio, sujeto + would/could/might + verbo base.

Ejemplo: If I had studied, I would know the answer now.

También: I would know the answer now if I had studied.

Mixed Conditional 2

Uso: una condición irreal del presente explica un resultado del pasado.

If + sujeto + past simple, sujeto + would/could/might + have + participio.

Ejemplo: If I were more careful, I would have avoided the mistake.

También: I would have avoided the mistake if I were more careful.

Regla importante: si la oración empieza con If, normalmente se usa coma. Si empieza con la oración principal, no se usa coma antes de if.
