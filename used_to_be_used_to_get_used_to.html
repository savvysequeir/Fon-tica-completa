II. Explicación

1. Used to + verb base

Se usa para hablar de hábitos, rutinas o estados del pasado que ya no son iguales en el presente.

Estructura: Subject + used to + verb base + complement.

Ejemplo: I used to wake up late. = Yo solía despertarme tarde.

2. Be used to + noun / verb-ing

Significa estar acostumbrado a una situación, cosa o actividad.

Estructura: Subject + am/is/are/was/were + used to + noun / verb-ing.

Ejemplo: She is used to studying at night. = Ella está acostumbrada a estudiar de noche.

3. Get used to + noun / verb-ing

Expresa el proceso de acostumbrarse a algo nuevo o difícil.

Estructura: Subject + get/gets/got/is getting + used to + noun / verb-ing.

Ejemplo: We are getting used to speaking English. = Nos estamos acostumbrando a hablar inglés.

Diferencia esencial

used to = solíabe used to = estar acostumbradoget used to = acostumbrarse

Cuidado: después de used to para hábito pasado va verbo base; después de be/get used to va sustantivo o verbo con -ing.
