II. Explicación

2.1 Comparativo de superioridad

Adjetivo corto: sujeto + be + adjective-er + than + objeto.

Adjetivo largo: sujeto + be + more + adjective + than + objeto.

Ana is taller than Luis. / This idea is more useful than that one.

2.2 Superlativo

Adjetivo corto: the + adjective-est.

Adjetivo largo: the most + adjective.

She is the tallest student. / This is the most important idea.

2.3 Igualdad

as + adjective + as expresa que dos elementos tienen el mismo nivel de una cualidad.

A dog is as loyal as a horse.

2.4 Desigualdad

not as + adjective + as o less + adjective + than expresan inferioridad o diferencia.

This phone is not as expensive as that laptop. / This option is less risky than the first one.

2.5 Reglas de escritura

CVC: big → bigger → the biggest. Y: easy → easier → the easiest. E final: large → larger → the largest. Irregulares: good → better → the best; bad → worse → the worst.

2.6 ¿Qué se puede comparar?

Personas, animales, objetos, ideas, lugares, acciones, decisiones, precios, niveles de dificultad, emociones y situaciones.
