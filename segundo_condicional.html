II. Explicación

2.1 Estructura

If + sujeto + pasado simple, sujeto + would/could/might + verbo base.

Ejemplo: If I studied more, I would pass the exam.

2.2 Positiva

Se usa pasado simple en la condición y would/could/might + verbo base en el resultado.

If she practiced, she would improve.

2.3 Negativa

En la condición: did not + verbo base. En el resultado: would not + verbo base.

If he did not study, he would not pass.

2.4 Interrogativa

Would/Could/Might + sujeto + verbo base + if + sujeto + pasado simple?

Would you travel if you had money?

2.5 Usos

Se usa para situaciones imaginarias, hipotéticas, poco probables o consejos indirectos en el presente o futuro.

2.6 Regla especial con be

En inglés formal se usa were con todos los sujetos: If I were you, I would study.
