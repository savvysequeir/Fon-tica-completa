II. Explicación

2.1 Estructura

If + sujeto + had + participio pasado, sujeto + would/could/might + have + participio pasado.

2.2 Positiva

If she had studied, she would have passed. También: She would have passed if she had studied.

2.3 Negativa

If he had not arrived late, he would not have missed the test.

2.4 Interrogativa

Would you have helped if you had known?

2.5 Usos

Se usa para hablar de situaciones imaginarias del pasado: arrepentimientos, críticas, consecuencias no realizadas o hipótesis imposibles porque ya pasaron.

2.6 Orden flexible

Puede empezar con If o con la consecuencia. Si empieza con la consecuencia, normalmente no lleva coma: I would have passed if I had studied.
